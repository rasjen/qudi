Detailed Description

Format code 'r' for scaled output.

Examples

'{:.0r}A'.format(ScaledFloat(50))

50 A

'{:.1r}A'.format(ScaledFloat(1.5e3))

1.5 kA

'{:.1r}A'.format(ScaledFloat(2e-3))

2.0 mA

'{:rg}A'.format(ScaledFloat(2e-3))

2 mA

'{:rf}A'.format(ScaledFloat(2e-3))

2.000000 mA
