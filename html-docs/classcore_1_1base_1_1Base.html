Detailed Description

Base class for all loadable modules.

* Ensure that the program will not die during the load of modules in any case, and therefore do nothing!!! * Initialize modules * Provides a self identification of the used module * Output redirection (instead of print) * Provides a self de-initialization of the used module * Reload the module with code changes * Get your own configuration (for save) * Get name of status variables * Get status variables * Reload module data (from saved variables)
